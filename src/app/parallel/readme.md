## Parallel Route

របៀបដែលយើងប្រើប្រាស់នៅ Parallel Route

យើងត្រូវមាននៅ `layout.tsx` ដើម្បីជា route មួយដែលហៅនៅ parallel ទាំងពី។ យើងត្រូវមាននៅ `page.tsx` ដែល render នៅ​ page ពីក្នុង ដើម្បីយកមក display

- ត្រូវប្រើប្រាស់នៅ file `layout.tsx` ដើម្បីបញ្ចាក់អំពី route
- រូវប្រើប្រាស់នៅ file `page.tsx` ដើម្បីបញ្ចាក់អំពី page ដែលនឹងត្រូវ render សម្រាប់ layout
